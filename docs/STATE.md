## Известные уроки и риски

- **Edge хвоста сжимается с OEV-recapture.** Morpho поддерживает pre-liquidations с
  кастомным оракулом (вкл. Chainlink SVR). По мере внедрения кураторами хвостовой
  бонус ужимается → «собирай, пока дают, оставайся мобильным по новым рынкам».
- **Класс багов, который py_compile пропускает** (перенесено из twidgest/essayist):
  имена внутри функций, локальный `from..import X` затеняющий модульное имя,
  несуществующий атрибут. AST-проверка обязательна.
- **Числа «до проверки» ≠ «итог».** Не показывать кандидатов как результат
  (карточный урок twidgest).
- **systemd `ProtectHome=true` ломает `/root`-сервис.** Юнит изначально имел
  `User=liquidator` + `ProtectHome=true` при `WorkingDirectory=/root/liquidator` —
  сервис не видел бы свою папку (поймано на скаффолде, до прода). Фикс: идём от root
  в `/root` (как twidgest), без `ProtectHome`. Не возвращать без переноса в `/opt`.
